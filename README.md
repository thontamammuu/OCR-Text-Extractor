# 📄 OCR Text Extractor using EasyOCR & Streamlit

## Project Description

This project is an AI-based OCR application that extracts text from images and PDF documents.

It uses EasyOCR for text recognition and Streamlit to create a simple web application.

## Features

- Upload images (JPG, JPEG, PNG)
- Upload PDF documents
- Extract text automatically
- Display extracted text

## Technologies Used

- Python
- Streamlit
- EasyOCR
- PyMuPDF
- Pillow
- NumPy
